🐍 3D Snake – Multi‑Language Edition
An immersive 3D Snake game where you control a snake navigating a cubic grid, collecting food, and avoiding collisions—all rendered in a 3D isometric perspective right in your terminal!
Built in 7 programming languages – perfect for learning 3D projections, game loops, and data structures.

✨ Features
3D isometric rendering – the game world is displayed in a 3D perspective using ASCII/block characters.

Smooth movement – the snake moves in 3D space (X, Y, Z axes) with intuitive controls.

3D food spawning – food appears at random positions in the 3D grid.

Scoring & speed – score increases with each food eaten; speed gradually increases.

Wall wrapping – snake wraps around the edges of the 3D world (optional, can be toggled).

High score tracking – stores the highest score achieved (per session).

Pause/Resume – pause the game at any time.

Cross‑platform – runs in any terminal (Windows, macOS, Linux).

🚀 How to Run
Each file is standalone – run it with the appropriate interpreter/compiler.

Language	Command
Python	python snake3d.py
Go	go run snake3d.go
JavaScript	node snake3d.js
C#	dotnet run (or csc Snake3D.cs && Snake3D.exe)
Java	javac Snake3D.java && java Snake3D
Ruby	ruby snake3d.rb
Swift	swift snake3d.swift
🎮 Controls
W / Arrow Up – move forward (Z+)

S / Arrow Down – move backward (Z-)

A / Arrow Left – move left (X-)

D / Arrow Right – move right (X+)

Q – move up (Y+)

E – move down (Y-)

Space – pause / resume

R – restart after game over

Esc – quit

📦 3D World
Grid size: 8×8×8 (configurable).

The snake starts at the center of the grid.

Food spawns randomly; eating it increases length and score.

The game ends when the snake hits itself or the world boundary (if wall wrapping is off).
